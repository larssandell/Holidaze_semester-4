### Installing

1. Clone the repo:

```bash
git clone https://github.com/larssandell/Holidaze_semester-4.git
```

2. Install the dependencies:

```
npm install

```

3. Build:

```
npm run build

```

4. Start ViteJS:

```
npm run dev
```

5. unit-test

```
npm run unit
```
